# LoRaWan_TTNUno_GY-BME280_I2C
This is a basic code for using the BMPE280 with The Things Uno.<ctrl>


The GY-BME280 is in I2C mode and has the following pinsetting:<ctrl>
  
GY-BME280_I2C <tab> The ThingsUno<ctrl>
  
Vcc <tab> 3.3V <ctrl>
Gnd <tab> Gnd <ctrl>
SCL <tab> SCL <ctrl>
SDA <tab> SDA <ctrl>
